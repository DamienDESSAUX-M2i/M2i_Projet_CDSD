Ce projet est un élément central du diplôme Concepteur·Développeur en Science des Données (Jedha – RNCP 35288) et doit démontrer que vous maîtrisez les compétences attendues par le référentiel officiel.
 
1. Ce que le diplôme Jedha attend concrètement
 
Un projet de soutenance doit montrer que vous êtes capables de :

- Concevoir une infrastructure de gestion de données (type data lake / data warehouse),
- Collecter des données depuis différentes sources,
- Nettoyer, transformer et organiser les données via des processus de type ETL,
- Stocker les données de manière fiable et exploitable,
- Développer un projet data structuré en Python,
- Industrialiser une solution (projet reproductible, déployable, maintenable),
- Gérer un projet data de bout en bout, dans une logique professionnelle.

2. Ce que vous devez mettre en place dès maintenant

En cohérence avec les modules que vous avez déjà vus, et avant même de figer votre sujet, je vous demande de poser une base générique de projet comprenant a minima :

- Une infrastructure conteneurisée (Docker / Docker Compose),
- Un système de stockage de données de type stockage objet (MinIO),
- Un pipeline ETL en Python (collecte, transformation, chargement),
- Un dépôt Git structuré, utilisé tout au long du projet.

L’objectif à ce stade est de poser une architecture propre, claire et professionnelle, conforme aux attentes du diplôme.
 
3. Un projet qui évoluera avec les modules

Votre projet n’est pas figé aujourd’hui.
Au fur et à mesure que nous avancerons dans les modules du cursus, vous serez amenés à :

- compléter votre sujet,
- enrichir votre architecture,
- intégrer de nouvelles briques (analyse, modélisation, visualisation, industrialisation, etc.).

Le projet de soutenance est donc progressif et se construit module après module, comme dans un vrai projet data en entreprise.
 
4. Aide à venir sur les sujets

Enfin, pour vous accompagner :

- d’ici quelques semaines, je vous proposerai plusieurs thématiques de projet,
- afin de vous donner des idées,
- de vous aider à cadrer un sujet réaliste,
- et de garantir l’alignement avec les exigences de la soutenance Jedha.
